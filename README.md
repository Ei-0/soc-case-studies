# SOC & DFIR Case Studies Portfolio

This repository contains practical security analysis case studies
focused on Security Operations Center (SOC) work and Digital Forensics
and Incident Response (DFIR) mindset.

## What this portfolio demonstrates
- Analytical thinking over tool usage
- Incident correlation and timeline reconstruction
- Decision-making under uncertainty
- Business impact awareness
- Clear and structured security documentation

## Case Studies
The case studies in this repository simulate real-world scenarios and
focus on understanding attacker intent, system behavior, and
appropriate response decisions.

> All scenarios are simulated for educational purposes. No real systems
or organizations are involved.
